# OctoAcme Project Management Docs — README

## What this is
A single landing page that summarizes OctoAcme's project management processes and links to the canonical docs in this folder.

## Summary of updates in this PR
This PR expands role definitions (adds Release Manager, Platform/SRE, Security Liaison, UX Researcher, Data Owner, and TPM), and introduces practical templates: a Release Checklist and a Dependency Register template. These changes aim to reduce ambiguity in releases, clarify cross-team responsibilities, and improve onboarding.
